Lab #9

My Favorite Icon

What is Favicon

Favicon is a small logo that represent a website. This icon displayed in various place of your website, In the top left next to the site name and also in the tab. This file save as .ico

How can we create a Favicon

To create a favicon first of all we all should know what our design going to be for our icon. Then we should know the size we going to be use. There's are so many icon creator website online that we can simply use and also we can use photoshop.

How I created my own Favicon

To create my own favicon I didn't think too much about design. I wanted to use it for my own website so I thought why not i just use my initial of my name. So at the top the icon you can see it represent my intial name "MI". So at first I had to decide what size I'm going to use where I used 20 by 20 for this icon. Later I made a draft how it's going to look. So I made my original icon into code.org website. There was option for table size and byte, where I used 6 bytes. And later I put the color code which suits for my icon.
